## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/JuanJo-R-1/Portafolio-JuanJo.git
   ```

2. **Open the project**
   - Open the folder in Visual Studio Code or your preferred editor
   - Install the Live Server extension if you're using VS Code

3. **Preview the site**
   - Right-click on `HTML/portafolio.html`
   - Select "Open with Live Server"
   - The site will open in your default browser

## Customization

The portfolio uses CSS variables for easy customization. Main colors and styles can be adjusted in `CSS/style.css`:

```css
:root {
    /*  Add your custom colors here */
    --primary-color: #00a8cc;
    --accent-color: #ff7e1f;
}
```
